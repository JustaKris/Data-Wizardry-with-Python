# Data Visualization

Learn to create compelling visualizations with matplotlib and seaborn.

## Visualization Libraries

Python has several visualization libraries:

- **matplotlib**: Low-level, highly customizable
- **seaborn**: High-level, statistical plots, beautiful defaults
- **plotly**: Interactive plots
- **pandas**: Built-in plotting (uses matplotlib)

This tutorial focuses on matplotlib and seaborn.

## Setup

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Set style
sns.set_style('whitegrid')
plt.rcParams['figure.figsize'] = (10, 6)
```
